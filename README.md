# Saynaa hobby os

This is a minimal 32-bit x86 hobby os

-----------------------------------------
Requirements

Tools needed:

- nasm               : Assembler
- qemu-system-i386   : Emulator
- make               : Build automation (usually preinstalled)
- clang              : C compiler
- grub               : Create bootable ISO with GRUB

-----------------------------------------
Build Instructions

Run:
```bash
make
```
-----------------------------------------
Run in QEMU

Run:
```bash
make qemu
```
